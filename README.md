# reproduce
* install module
  * opencv
  * pyqt5
  ```
     pip install PyQt5
  ```
  * imutils
  ```
     pip install imutils
  ```
  * face_recognition
  ```
     pip install face-recognition
  ```
  * dlib
  ```
     pip install dlib
  ```
* 下載好想要p圖的兩張照片
   (預設照片存於Downloads資料夾)
* 在project1裡，創建5個新資料夾：
  * picture
  * blink1
  * blink2
  * no_blink1
  * no_blink2
* run gui.py
